计算选择器的特指度

主要针对这种情况：当一个元素被多条 CSS 选择器选中时。哪一条 CSS 选择器所指定的规则会生效，就要取决于它的选择器有多特定，也就是特指度是多少，可以使用下面的 "ICE" 公式计算三个值。

针对这个公式的计分方法

  1. 选择符中有一个 id, 就在 I 的位置加上 1
  2. 选择符中有一个 class, 就在 C 的位置加上 1
  3. 选择符中有一个元素 (标签) 名，就在 E 的位置加上 1
  4. 得到 "I-C-E" 格式的数字

简单示例

  • p
    • 0-0-1
  • p.largetext
    • 0-1-1
  • p#largetext
    • 1-0-1
  • body p#largetext
    • 1-0-2
  • body p#largetext ul.mylist
    • 1-1-3
  • body p#largetext ul.mylist li
    • 1-1-4

最后

如果特指值相同，则后面的规则会覆盖之前的规则。
