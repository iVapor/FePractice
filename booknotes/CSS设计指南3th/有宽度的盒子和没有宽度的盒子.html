有宽度的盒子和没有宽度的盒子

有宽度和没有宽度指的是元素有没有声明 width 属性，盒子指的是盒模型。

没有宽度的盒子

我是一个没有宽度的盒子，字都是为了凑字数，来达到效果。

开始变身，添加属性：padding: 0 20px; 文本内容的宽度减少了。

我是一个没有宽度的盒子，字都是为了凑字数，来达到效果。

再次变身，添加属性：border: solid red; border-width: 0 6px 0 6px; 文本内容的宽度再次减少。

我是一个没有宽度的盒子，字都是为了凑字数，来达到效果。

变身完成，添加属性：margin: 0 30px; 文本内容的宽度还是会减少。

我是一个没有宽度的盒子，字都是为了凑字数，来达到效果。

小结

没有指定宽度的块元素，它的宽度始终是填满父元素的，即使我们添加了 padding, border, margin 属性，不会影响元素的宽度，但是会影响元素中的内容区的宽度。

有宽度的盒子

我的宽度是 400px，字都是为了凑字数，来达到效果。

开始变身，添加属性：padding: 0 20px; 文本内容的宽度居然增加了。

我的宽度是 400px，字都是为了凑字数，来达到效果。

再次变身，添加属性：border: solid red; border-width: 0 6px 0 6px; 文本内容的宽度再次增加。

我的宽度是 400px，字都是为了凑字数，来达到效果。

变身完成，添加属性：margin: 0 30px; 文本内容的宽度还是会增加。

我的宽度是 400px，字都是为了凑字数，来达到效果。

小结

有指定宽度的块元素，它的宽度其实指的是内容区的宽度，如果这时候添加 padding, border, margin 属性，会使元素的整体宽度增加。
